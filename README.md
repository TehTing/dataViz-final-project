Youtify-Conbine Spotify with Youtube
===

## 基本介紹
本專案嘗試彙整Kaggle平台中Spotify 2010-2019間的單曲統計資料，並以資料視覺化的方式呈現
呈現類型如下:
### 1. 分類
  * Pop
  * Genre
  * Artist
### 2. 圖形
  * Bar chart
  * Pie chart 
## 創意點綴
單純的資料視覺化可能太單純，因此團隊嘗試讓user聆聽圖中所對應的音樂，當點選的Bar(Pie) chart中的 bar/pie時，
會在下方資訊欄提供youtube影片，數各位閱聽者享受愉快，但也請注意有點選次數限制唷!!
